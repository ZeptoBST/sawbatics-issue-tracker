# sawbatics-issue-tracker
This repository serves to track bugs in Secret Agent Wizard Boy and the International Crime Syndicate.

Here are the different labels to use:
--------------------------------------------------------------

Actors =     When a bug relates to anything around npcs.

Animations =     When a bug relates to character animations.

Audio =     Anything related to audio, wether it's music or sfx.

Collision =     Anything related to model/object collision.

Enhancement =     For suggestions that can improve the game.

Events =     If a bug is related to an event supposed to happen in-game (ex: the alarm going off when commiting a crime).

FPS =     Anything that has an effect on frame rate.

Game Breaking =     For bugs causing serious issues in the game, such as alting progression.

Graphics =     Anything related to visuals/what you can see on-screen (ex: models, effects, particles...).

Input =     Anything related to controls, wether it's Keyboard and mouse or controller.

Likely Won't Change =     When a bug is not going to be fixed.

Missions =     When a bug affects missions in the game (ex: shape key spawning in wrong location).

Multiplayer =     Bugs that occur exclusively in multiplayer mode.

Objectives =     Anything related to in-game achievements.

Objects =     Anything related to objects; anything from big objects (ex: cabinet, shopping kart, barrel...) to small objects (ex: apple, vase, breadstick...).

Pending =     When a bug is set to be fixed for a later build.

Technical =     Specifically for technical issues (ex: game crashes).

UI =     Anything related to on-screen elements (ex: hud, menu, control prompts...).
